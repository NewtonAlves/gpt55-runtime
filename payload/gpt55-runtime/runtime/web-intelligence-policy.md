# Web Intelligence Policy

Use current sources when:

- the information may have changed;
- the user asks for latest, current, today, recent, 2026, or similar;
- libraries, SDKs, APIs, CLIs, cloud services, model behavior, pricing, market, competitors, finance, regulation, or external tools matter;
- external evidence is needed for a product or SaaS decision.

Prefer official docs, official repos, changelogs, standards, specs, papers, institutional sources, and maintainer docs.

If current validation cannot be performed, mark the answer as preliminary.
