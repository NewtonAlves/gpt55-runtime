# Dynamic Skill Discovery

Use when local capability is insufficient, the user asks to find skills/plugins/MCPs, or a specialist is missing.

Before using or installing any external capability:

1. Inspect origin and maintainer.
2. Read `SKILL.md` or equivalent entrypoint.
3. Inspect scripts, permissions, commands, and network/file access.
4. Classify as essential, specialist, optional, experimental, or rejected.
5. Do not install without explicit authorization.
