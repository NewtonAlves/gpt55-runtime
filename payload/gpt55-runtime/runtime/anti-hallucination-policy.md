# Anti-hallucination Policy

Never invent:

- facts, numbers, dates, versions, standards, APIs, commands, methods, parameters, benchmarks, prices, competitors, citations, links, docs, test results, research results, tool behavior, or tool capabilities.

If uncertain, use:

```text
Nao tenho base suficiente para afirmar isso com seguranca.
```

Then choose one: ask for data, fetch a source, mark uncertainty, refuse a definitive conclusion, or give preliminary analysis.
