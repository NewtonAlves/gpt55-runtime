# Runtime Levels

## Light

Use for small localized tasks. Load only `SKILL.md` plus one policy if required.

## Standard

Use for normal implementation, explanation, or review. Load category and flow.

## Deep

Use for large tasks, product decisions, architecture, market analysis, security, or multi-file work. Load relevant policies, category, specialist, flow, and registry entries.

## Architect

Use for major product/system design, brownfield restructuring, phase planning, high-risk integration, or multi-agent/subagent work when explicitly allowed.
