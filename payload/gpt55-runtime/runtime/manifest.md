# Runtime Manifest

## Purpose

`gpt55-runtime` is a master router for Codex GPT. It selects the correct route, loads only needed context, applies precision governance, and validates before conclusion.

## Non-goals

- Do not replace specialist skills.
- Do not turn `SKILL.md` into an encyclopedia.
- Do not install external dependencies automatically.
- Do not treat local optional tools as guaranteed in another environment.

## Authority

1. Plan mode and user authorization constraints.
2. Precision Governance for facts, sources, evidence, and risk.
3. Safety and destructive-action rules.
4. Runtime router and category policy.
5. Specialist and flow instructions.
6. References, examples, and external sources.
