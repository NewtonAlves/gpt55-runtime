# Category Taxonomy Summary

The runtime uses categories for routing, specialists for precision, flows for repeatable work, registries for lookup, and references for large supporting material.

Core category families:

- engineering and code;
- frontend/design;
- backend/data/auth;
- mobile;
- gamedev;
- AI agents/MCP;
- education;
- financial/business/product;
- security/devops;
- writing/books;
- research/discovery.

Do not create new categories in v1 unless an approved plan requires it.
