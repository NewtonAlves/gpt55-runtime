# Skill Sources Guide

Use official skill catalogs and docs first. Treat marketplace or community skills as third-party code.

Audit before use:

- source and maintainer;
- `SKILL.md` content;
- scripts and commands;
- file/network access;
- install instructions;
- overlap with existing skills;
- maintenance status;
- operational risk.

Classification: essential, specialist, optional, experimental, rejected.
