# Update Process Summary

For skill/archive updates:

1. Identify target capability.
2. Inspect existing local capability.
3. Audit source if external.
4. Ask explicit authorization before installing anything new.
5. Update modular file, registry, and validation test.
6. Keep `SKILL.md` short.
7. Validate references, stale paths, duplicates, and routing.
8. Report changed files and residual risk.
