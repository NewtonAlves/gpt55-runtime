# DevOps Category

Use for CI/CD, Docker, deploy, infrastructure, monitoring, local services, cloud configuration, and release process.

Aliases: devops, docker, deploy, ci, cd, infra, cloud.

Specialists: `critic-reviewer`.

Flows: `backend-api-flow`, `example-driven-development-flow`.

Cost: medium; high for production, secrets, destructive commands, or infrastructure changes.

Validation: prefer dry-run or read-only checks before mutating infrastructure.
