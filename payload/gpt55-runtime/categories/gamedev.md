# Gamedev Category

Use for game design, 2D/3D games, Godot, Phaser, Unity, Unreal, gameplay loops, assets, and playtesting.

Aliases: game, gamedev, godot, phaser, unity, unreal.

Specialists: `critic-reviewer`.

Flows: `gamedev-flow`, `example-driven-development-flow`.

References: gamedev curadoria and dev examples.

Cost: medium; high for 3D, asset pipelines, or playtesting.

Validation: run the game, check rendering, input, win/loss states, and performance basics.
