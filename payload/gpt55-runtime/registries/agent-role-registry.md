# Agent Role Registry

Use subagents only when explicitly allowed.

| Role | Use | Constraints |
|---|---|---|
| researcher | Independent source gathering | Must cite sources and uncertainty |
| implementer | Disjoint code edits | Must not overwrite others' work |
| reviewer | Quality/security review | Findings first |
| verifier | Acceptance validation | Use evidence, not assumptions |
| critic | Challenge plan or output | Must be grounded in artifacts |
