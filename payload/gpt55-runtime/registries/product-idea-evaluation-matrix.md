# Product Idea Evaluation Matrix

Score each dimension from 0 to 3:

| Dimension | 0 | 1 | 2 | 3 |
|---|---|---|---|---|
| Problem clarity | vague | named | specific | painful and repeated |
| User definition | none | broad | segment | reachable buyer/user |
| Evidence | none | anecdotal | external signals | strong current evidence |
| Competition | unknown | crowded/unclear | mapped | clear wedge |
| Monetization | none | speculative | plausible | validated signal |
| Buildability | unclear | hard | feasible | small MVP possible |
| Distribution | none | weak | plausible | clear channel |
| Risk | severe | high | manageable | low/known |

Classification:

- 18+ Evidence-backed if sources support it.
- 12-17 Plausible.
- 7-11 Speculative.
- 0-6 Avoid unless new evidence appears.
