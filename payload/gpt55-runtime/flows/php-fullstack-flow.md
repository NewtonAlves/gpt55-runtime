# PHP Fullstack Flow

## Objective

Create, review or refactor PHP + HTML + CSS + JavaScript projects without a heavy framework, prioritizing modern PHP, PDO, OWASP-aware security, semantic HTML, responsive CSS and vanilla JavaScript.

## Sequence

1. Understand scope: CRUD, login, dashboard, form, API endpoint, upload, mini app, plain PHP or simple MVC.
2. Choose architecture: simple project, `public/`, `src/`, `views/`, `config/`, minimal MVC or PHP endpoint plus Fetch.
3. Verify environment: PHP version, MySQL/MariaDB, local server option and Composer when needed.
4. Define structure: `public/`, `src/`, `views/`, `config/`, `storage/`, `README.md` and `.env.example`.
5. Implement PHP backend with PDO, prepared statements, server-side validation and safe error handling.
6. Implement security with safe password hashing, session regeneration, CSRF, output escaping, upload validation and secure cookies when applicable.
7. Implement frontend with semantic HTML, accessible forms, mobile-first CSS, visible focus, `addEventListener`, Fetch and loading/error/success states.
8. Validate no SQL concatenation, plaintext passwords, sensitive localStorage, unsafe `innerHTML`, unsafe uploads or missing setup docs.
9. Deliver summary, changed files, how to run, tests/checks, residual risks and next steps.
